### Description
---

Collision Fix which fixes client-side collisions checks to prevent you from pushing by other entities. Mobs simply won't be able to push you if they get too close. **You can still be pushed with normal attacks, etc.**

![Showcase](https://cdn.modrinth.com/data/CBtq0jWL/images/997ec7add18e009f0ff8af43a546810ce0da1301.gif)

### Configurable
---

Starting with newer versions of the mod, you can choose which entities are able to push you. You can also enable this only on specific servers.
